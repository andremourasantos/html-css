Usandos links

Links externos

Você pode usar links para outros sites fora do seu de duas maneiras diferentes:

  1. Fazendo com que o link externo se sobreponha ao seu site.
  2. O site externo abre em uma nova guia no nnavegador.

Para o seu site, é extremamente recomendável que você utilize o segundo método, pois dessa maneira o cliente ainda terá o seu site aberto, coisa que no primeiro método não acontece.

  • Veja dois exemplos de aplicação:
    1. Clique nesse link para abrir o YouTube.
    2. Clique nesse link para abrir o Youtube, mas em outra guia do navegador.

Links para download

Você pode criar links para donwload de arquivos no seu site utilziando a mesma tag <a> que anteriormente, apenas adicionando os atributos de download="[nomedoarquivo].pdf" e target="application/pdf", por exemplo.

Para os parâmetros do atributo target, você pode conferir o site da IANNA sobre Media Types.
