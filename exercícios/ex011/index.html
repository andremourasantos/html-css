Imagens dinâmicas

Teste aumentar e diminuir o tamanho da janela do seu navegador e veja a imagem ser alterada para uma de tamanho correspondente!

Para configurar essa mudança utiliza-se a tag <picture>. Essa tag possibilita o carregamento de diferentes imagens dependendo de padrões pré-estabelecidos, que no caso é o de tamanho da janela do navegador.

O código final ficou dessa maneira: 


    <picture>
        <source media="(max-width: 750px)" srcset="imagens/foto-p.png" type="image/png">
        <source media="(max-width: 1050px)" srcset="imagens/foto-m.png" type="image/png">
        <img src="imagens/foto-g.png" alt="Foto grande" type="image/png">
    </picture>

Os atributos "source media" estão sendo utilizados, em ordem hierárquica, de cima para baixo em prioridade, para definir que imagem deve ser carregada baseando-se no tamanho da janela aberta.

Note que há três tamanho de imagem: A imagem carregada por padrão é a maior (foto-g.png), ela é a imagem da tag <img src=...>. A foto-g.png tem um tamanho de 1000px, portanto, caso a janela seja menor que 1050px (adicionou-se uma margem de 50px na conta), a imagem que deve abrir não é mais a foto-g.png, e sim uma menor, a foto-m.png, que contém 700px. O mesmo raciocínio segue para a exibição da foto-p.png.
