Outras formatações

Código fonte / Pré formatação

É possível utilizar a tag <code> para representar linhas de códigos no seu site elas fiarão exatamente desse jeito.

Também é possível usar a tag <pre> para representar o código da exata maneira que ele foi escrito no código HTML da página. 

                
                    Então ele aparecerá assim,
                    exatamente da mesma maneira
                    que
                    ele foi escrito

        desculpe escrever tudo torto assim kkk

Citações simples

Você pode usar a tag <p> para adicionar citações curtas ao seu texto, como por exemplo: João foi ao parque sexta-feira, ao menos foi o que o pai dele me disse.

Citações completas

Também tem como adicionar citações mais complexas, bem no estilo dos padrões da ABNT mesmo.

"Nesse caso aqui eu não tenho nenhuma citação pra colocar, mas é assim que ela ficaria".
